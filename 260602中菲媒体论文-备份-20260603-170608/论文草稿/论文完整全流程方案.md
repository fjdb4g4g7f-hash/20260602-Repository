## 第二部分：研究问题与假设

### 2.1 三个研究问题（RQ）

**RQ1（议题维度）**：中菲官方媒体在南海议题的系统叙事上呈现出怎样的结构化差异？双方如何通过选择性议题框定来建构对立的区域秩序认知？

**RQ2（身份维度）**：中菲官方媒体如何通过语法角色配置与行动者网络建构"自我"与"他者"的身份叙事？这种身份建构呈现出怎样的非对称性？

**RQ3（系统维度）**：中菲媒体对"安全""和平""主权""规则"等核心概念是否存在语义分歧？这种语义分歧如何反映双方在区域秩序认知上的底层分歧？

### 2.2 三个研究假设（H）

**H1（议题框定假设）**：中方媒体将南海议题框定为"合作治理/区域发展"叙事，菲方媒体框定为"冲突/法律化/同盟"叙事，二者议题结构呈镜像对立，且差异在统计上显著。

**H2（身份建构假设）**：中方媒体建构"区域和平维护者"的自我身份与"域外干预力量"的他者身份；菲方媒体建构"主权受害者"的自我身份与"秩序破坏者/侵略者"的他者身份。双方身份互动呈非对称的"合作vs抗议"模式。

**H3（语义分歧假设）**：中菲媒体对核心概念（security/peace/sovereignty/rules/order）存在系统性语义分歧，表现在不同的近义词场和搭配模式上，语义场重叠度显著低于随机水平。

### 3.2 三个理论如何对应三个分析方法

| 傅强路径 | 我们的方法 | 对应Miskimmon的叙事维度 | 对应曹德军的解释机制 |

| **词语路径** (启动效应+累积效应) | BERTopic主题结构 + 词频/关注度分析 | 议题叙事(What issue) | 框定机制(选择性凸显) |

> **核心优势**：傅强(2024)的"威胁建构三路径"框架天然地与我们的三种计算方法一一对应，不需要任何强行匹配。词语路径→词频/主题分析、隐喻路径→语义场分析、叙事路径→SVO分析，这是自然的方法论契合。

### 写作避坑指南

| 坑 | 避免方式 |
|----|---------|
| 引言写得像文献综述 | 引言仅聚焦"问题是什么"，文献回顾单独成节 |
| 方法部分堆砌术语 | 每个方法用一句话说清楚"为什么用这个方法" |
| 数据分析像方法展示 | 每个分析结果都要回扣到理论+研究问题 |
| 讨论变成"重复结果" | 讨论要解释"这个发现意味着什么"，而非"这个发现了什么" |
| 结论过度拔高 | 结论只讲"本文做了什么、发现了什么、贡献了什么" |
| 引用泛滥 | 只引最核心的20-25篇文献，重质不重量 |
| 安全化理论的残留术语 | 全文搜索"安全化/去安全化/反安全化/话语安全困境"等删除 |
| IR色彩过重的表达 | 将"权力竞争/安全困境/战略博弈"改为"叙事竞争/话语张力/框定博弈" |
