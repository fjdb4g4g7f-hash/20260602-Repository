## 一、你的四个核心疑虑 — 逐一分析

### 疑虑1：安全化理论的适用性问题 ⭐⭐⭐⭐⭐

**你的判断**：
> "安全化理论用于将非传统安全议题建构为安全问题。南海问题属于原生安全议题，并非'被安全化'的对象。该理论在本文中的应用合理性、有效性需重新验证。"

**我的分析：你的判断基本正确，这是一个非常尖锐且有效的理论批评。**

哥本哈根学派的"安全化"（Securitization）确实有一个核心前提：被安全化的对象原本是"非安全化"的议题（如移民、气候、传染病），通过言语行为（speech act）被建构为"生存性威胁"，从而获得采取非常规措施的正当性。

南海争端的特殊性在于：
- **它本质上是主权/领土争端**——从一开始就是"原生安全议题"（primary security issue），不是被话语建构出来的安全威胁
- 中菲海上摩擦涉及军舰、海警、渔民、岛礁基建——这些都是**物质性的安全现实**，而非纯粹的话语建构
- 安全化理论解释的是"非安全→安全"的转化过程，而南海议题一开始就在安全领域内

**但这不是说安全化理论完全不能用。** 可以两条路径处理：

**路径A（推荐）：将"安全化"改为更准确的替代概念**
- "话语博弈"（discursive contestation）
- "叙事对抗"（narrative confrontation）
- "威胁话语建构"（threat discourse construction）——注意这里是"威胁"被建构，而非"安全议题"被建构
- "话语极化"（discursive polarization）
- 这些概念不需要安全化理论的"非安全→安全"前提，但同样可以描述双方的叙事对抗升级过程

**路径B（如果你不愿完全放弃安全化理论）：**
- 可以保留但严格限定使用范围：安全化不是指南海争端"变成"安全议题，而是指双方在微观层面将**具体的海上事件**（如一次补给行动、一次对峙）建构为不同性质的威胁
- 引用Balzacq的"受众中心安全化"——强调安全化成功与否取决于受众接受度
- 但这需要在论文中做大幅度的理论重新定位

**我的推荐：路径A，将安全化理论替换为更贴切的话语博弈框架。** 这样既避免了理论适用性的争议，也让论文更清晰。

### 疑虑2：竞争论文的撞稿风险 ⭐⭐⭐⭐

**竞争论文**：梁昊光、秦清华，"浪潮与锚链：中美海上竞争的战略场域与话语分野"，《太平洋学报》2026年第2期

**逐项对比**：

| 维度 | 你的论文 | 竞争论文 |
|------|---------|---------|
| **研究对象** | 中菲（CN-PH） | 中美（CN-US） |
| **数据类型** | 六家主流英文媒体报道 | 官方外交/战略文本 |
| **时间跨度** | 2022.06-2025.12 | 2021-2025 |
| **理论框架** | 战略叙事+安全化 | 战略场域理论（布迪厄）+安全化 |
| **核心方法** | BERTopic主题建模 | 结构主题模型（STM） |
| **语义分析** | 词嵌入语义网络 | 历时词嵌入+情感分析 |
| **动态分析** | ✅ 事件驱动的语义距离追踪 | ❌ 无 |
| **发表状态** | 未投稿 | 已发表（2026年2月） |
| **学科方向** | 新闻传播/国际传播 | 国际关系/计算政治学 |

**撞稿风险评估：中低风险**

核心原因：
1. **研究对象不同**：中美 vs 中菲是不同的战略关系，分析逻辑完全不同。你的论文核心发现（中菲三维叙事错位、非对称安全化互动）不可能出现在中美论文中
2. **数据类型不同**：媒体报道 vs 官方外交文本。媒体报道更丰富、更多元，能反映公共舆论层面的话语博弈，这是官方文本做不到的
3. **你有独特的动态事件分析**：事件驱动的语义距离追踪是你论文的最大亮点，竞争论文完全没有这个维度
4. **理论框架可以进一步区分**：如果你按疑虑1走路径A（放弃安全化理论），那与竞争论文的重合度会进一步降低

**需要注意的地方**：
- 竞争论文也使用了"安全化"概念，所以你与他们共享了这个理论标签
- 竞争论文作者是中科院团队+国家社科重大项目，学术影响力较强
- 如果你的论文投稿到国际关系类期刊（如《太平洋学报》本身），确实容易撞车
- **如果你投新闻传播类期刊，撞稿风险极低**

**结论**：竞争论文的存在恰恰支持了你的方向判断——**转投新闻传播领域**是对的。新闻传播期刊的审稿人不太可能拿你的论文和一篇发表在IR期刊上的中美论文做直接比较。而且新闻传播领域更看重媒体话语分析的方法创新，这正是你的强项。

### 疑虑4：理论框架精简优化 ⭐⭐⭐⭐

**你的方案**：仅保留战略叙事理论，放弃安全化理论

**我的分析：这个精简方向非常正确，建议进一步明确如下**：

**保留的核心框架**（Miskimmon + 曹德军）：

```
战略叙事理论
├── 系统叙事 (System Narratives)
│   └── 对应论文：区域秩序认知 → 多边规则 vs 同盟对冲
├── 身份叙事 (Identity Narratives)  
│   └── 对应论文：自我/他者角色建构 → 和平维护者 vs 主权受害者
└── 议题叙事 (Issue Narratives)
    └── 对应论文：具体事件性质界定 → 合法执法 vs 生存性威胁
```

**三层分析的逻辑链**：
1. **微观层**（词语）：核心概念的语义不可通约性 — 同一词在中菲媒体中承载不同语义
2. **中观层**（主题）：三维战略叙事的结构性错位 — 系统/身份/议题叙事的全面差异
3. **宏观层**（动态）：事件驱动的叙事博弈演变 — 关键事件如何冲击叙事错位

**需要从论文中删除/替换的概念**：
- ~~安全化 (Securitization)~~ → "叙事对抗升级" / "威胁话语建构"
- ~~去安全化 (Desecuritization)~~ → "合作叙事" / "去冲突化话语"
- ~~反安全化 (Counter-securitization)~~ → "反叙事" / "话语解构"
- ~~安全化受众~~ → "叙事受众" / "传播受众"
- ~~话语安全困境~~ → "叙事错位困境" / "话语对抗困境" / "传播认知闭环"
- ~~非常规措施~~ → 删除或改为"激进政策选项"

**理论框架的优势**：
- 一个理论（战略叙事）解释一个问题（叙事体系建构与语义分歧），更加清晰
- 三个维度已经足够支撑全文的分析层次
- 避免了安全化理论的争议和IR范式的包袱
- 更贴近新闻传播学科的问题意识
