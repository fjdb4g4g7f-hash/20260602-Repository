## 二、本次对话核心成果

### 决策1：理论框架换心 ✅ 已确认

**问题**：论文原用"战略叙事理论 + 安全化理论"双框架，但安全化理论适用性存疑。

**原因**：哥本哈根学派的安全化理论用于将**非传统安全议题**建构成安全问题。但南海争端本身就是主权/领土安全问题（原生安全议题），并非"被安全化"的对象。该理论在本文中逻辑不自洽，会议上已被多次质疑。

**决定**：**放弃安全化理论，仅保留战略叙事理论**（Miskimmon + 曹德军）。

**术语替换对照**：
| 原术语 | 替换为 |
|--------|--------|
| 安全化 (Securitization) | 威胁叙事建构 / 对抗性话语动员 |
| 去安全化 (Desecuritization) | 合作叙事 / 去冲突化话语 |
| 反安全化 (Counter-securitization) | 反叙事 / 话语解构 |
| 话语安全困境 | 叙事错位困境 / 话语对抗闭环 |
| 安全化受众 | 叙事受众 / 传播受众 |
| 非常规措施 | 删除 |

**新理论框架**：
- 唯一理论：战略叙事理论
- 三维度：系统叙事 + 身份叙事 + 议题叙事（Miskimmon）
- 三层分析：微观（词语语义）→ 中观（主题建模）→ 宏观（事件动态演化）

---

### 决策2：研究方向转向新闻传播 ✅ 已确认

**决定**：从国际关系方向转向**新闻传播学 → 国际传播 / 计算传播学**。

**核心命题**：国际传播中的概念重构与叙事体系建构

**理由**：
1. 数据是媒体报道——传播学核心数据，非IR首选
2. 方法链（BERTopic + 语义网络 + 事件分析）在传播学是方法创新
3. 战略叙事理论天然跨传播学和IR，传播学定位更自然
4. 规避IR领域的竞争论文（梁昊光，2026）
5. 新闻传播期刊（《新闻与传播研究》《国际新闻界》）对计算方法接受度上升

**目标期刊**：《新闻与传播研究》或《国际新闻界》

---

### 决策3：菲律宾数据源替换为官方来源 ✅ 已确认

**问题**：原论文中方媒体是官方/半官方（人民日报、中国日报、环球时报），菲方是商业媒体（问询者报、星报、马尼拉公报）。会议中多次被批评"媒体体制不对称导致结论偏差——中国官方媒体天然克制，菲律宾商业媒体天然追逐冲突"。

**决定**：菲方全部替换为官方来源，实现**官方对官方**的清洁对比。

**新菲方来源**：PNA（菲律宾国家通讯社）+ PIA（总统府信息局）+ DFA（外交部，但只有5条）

**中方来源不变**：China Daily + Global Times + People's Daily

---

### 决策4：不加第三国 ✅ 已确认

**决定**：不加美国或东南亚国家。原因：
- 加美国：变成三方比较，论文要重写；梁昊光那篇已做中美比较，撞稿风险加大
- 加东南亚：语言障碍、数据获取难、稀释中菲双边焦点
- 核心策略：修复中菲媒体类型不对称即可，不引入新变量

---

### 决策5：竞争论文风险评估 ✅ 已完成

**竞争论文**：梁昊光、秦清华，"浪潮与锚链：中美海上竞争的战略场域与话语分野"，《太平洋学报》2026年第2期

**对比结论**：撞稿风险中低。核心差异：
- 对象不同：中美 vs 中菲
- 数据不同：官方外交文本 vs 媒体报道
- 方法差异：对方无事件驱动的动态分析（这是你论文的核心亮点）
- 转到传播学期刊后进一步错开
